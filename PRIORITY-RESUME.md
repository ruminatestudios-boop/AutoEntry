# Priority when we next resume

**Order of work:**

1. **Get the product tighter** – Polish the core product (flow, copy, behaviour).
2. **UI and screens all sorted** – Finish and align all screens, especially end screens (success, errors, empty states). One consistent experience across the flow.
3. **Do Etsy as the first end-to-end flow** – No app-approval block (unlike Shopify). Nail: Connect Etsy → scan → review (with Etsy fields: tags, category) → Publish to Etsy → end screen. Test and polish the full journey.
4. **Then replicate for the rest** – Use that Etsy flow as the template for Shopify (once approved), eBay, and (later) TikTok Shop / Amazon. Same patterns; only connect endpoints and platform-specific form fields change (e.g. condition for eBay).

**Remember:** Don’t spread effort across all platforms at once. Finish one flow end-to-end (Etsy first, no block), then apply it to the rest.
